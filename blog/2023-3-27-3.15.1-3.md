# 3.15.1-3 更新

## [3.15.3] - 2023-3-27
- 新增: `tsconfig-base.json` `tsconfig-app.json` `tsconfig-module.json` 预设文件
  - 开发项目时候可以直接继承预设文件

```json title="tsconfig.json"
{
  "extends": "zenweb/tsconfig-app",
  "compilerOptions": {
    "outDir": "./app"
  },
  "include": ["src/**/*"]
}
```
