# inject - 依赖注入

依赖注入支持

## 依赖模块

无

## 配置项

此模块没有配置项

## Core 挂载项

| 挂载项 | 类型 | 功能 |
| ----- | --- | ---- |
| injector | Injector | singleton 作用域注入管理器

## Context 挂载项

| 挂载项 | 类型 | 功能 |
| ----- | --- | ---- |
| injector | Injector | request 作用域注入管理器

## 装饰器

| 名称 | 功能 |
| --- | --- |
| @Injectable | 标记类为可注入，支持构造参数自动注入
| @Init | 初始化方法，在依赖注入完成后自动调用
| @Inject | 属性注入

## 全局模式

| 方法 | 功能 |
| ----- | ---- |
| getInstance(target: Class) | 取得对象实例，对象实例会优先从 request 层获取，没有请求上下文则从 singleton 层获取
